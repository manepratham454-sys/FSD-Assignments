## 🎯 WHAT HAPPENS WHEN YOU RUN IT

```
Step 1: npm start
  → Starts Express server on port 3000
  → Connects to MongoDB
  → Sets up all API routes
  → Ready to accept requests

Step 2: npm run seed
  → Connects to MongoDB
  → Adds 6 sample travel packages
  → Ready for test data

Step 3: Open http://localhost:3000
  → Website loads
  → JavaScript fetches packages from API
  → Displays all 6 packages
  → Users can book and contact

Step 4: User Books Package
  → Frontend sends POST request
  → Backend saves to MongoDB
  → User gets confirmation

Step 5: User Submits Contact
  → Frontend sends POST request
  → Backend saves to MongoDB
  → User gets thank you message
```
